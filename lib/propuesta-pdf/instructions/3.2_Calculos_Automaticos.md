### Fórmula general

- Horas por actividad × tarifa del rol responsable = Subtotal por línea.
- Suma de todas las líneas = `Subtotal_USD`.
- `IVA_15` = `Subtotal_USD × 0.15`.
- `Total_Final_USD` = `Subtotal_USD + IVA_15`.

### Mapeo fijo actividad → rol responsable

Se usa el mismo mapeo de la Sección 7 de 3.1. No cambiar.

### Formas de pago según Subtotal

> **Regla LA-013 (crítica):** los porcentajes se calculan **siempre sobre el `Subtotal_USD` (sin IVA)**. El IVA 15 % se suma al monto de cada hito en el momento de emitir la factura. El texto de Forma de Pago siempre dice "sin IVA" explícitamente.

| Rango del Subtotal (USD) | Esquema | Detalle (montos sobre Subtotal sin IVA) |
|---|---|---|
| Menos de $5,000 | 1 pago | 100 % al inicio |
| Entre $5,000 y $15,000 | 2 pagos | 50 % al inicio · 50 % contra entrega |
| Más de $15,000 | 3 pagos | 30 % al inicio · 30 % a la firma del acta de pruebas · 40 % a la entrega final |

## Base de datos Líneas de Cotización

> Una fila por cada Actividad de la Sección 7 (3.1). Nueve filas por propuesta. No crear duplicados ni filas extra.

**Estructura de relaciones:**

Tarifario Corporativo → (Relation: Rol) → Líneas de Cotización ↔ Propuestas Finales

### Propiedades de la BD

| Propiedad | Tipo | Notas |
|---|---|---|
| `Descripcion_Linea` | Title | Ej: "Arquitecto — Toma de requerimientos" |
| `Propuesta` | Relation | → BD Propuestas Finales |
| `Rol` | Relation | → BD Tarifario Corporativo |
| `Costo_Hora_Rollup` | Rollup | Trae `Costo/Hora Base (USD)` desde el Tarifario |
| `Horas_Estimadas` | Number | Copiadas 1 a 1 desde la tabla de la Sección 7 de 3.1 |
| `Subtotal_Linea` | Formula | `prop("Costo_Hora_Rollup") * prop("Horas_Estimadas")` |
| `Fase_Proyecto` | Select | Toma de requerimientos / Diseño técnico / Desarrollo Backend / Desarrollo Frontend / Pruebas unitarias / Pruebas UAT / Despliegue / Capacitación / Documentación |
| `Notas` | Text | Justificación y referencia al proyecto histórico similar |

> Regla obligatoria: toda propuesta debe tener al menos una línea con `Fase_Proyecto = Pruebas unitarias` y otra con `Fase_Proyecto = Pruebas UAT`, ambas con `Horas_Estimadas > 0`. Si alguna de las dos está en 0, la propuesta no debe exportarse.

## Fórmulas y rollups en Propuestas Finales

### Rollup: Total de Horas

| Configuración | Valor |
|---|---|
| Nombre | `Total_Horas` |
| Tipo | Rollup |
| Relación | Líneas de Cotización (propiedad `Propuesta`) |
| Campo | `Horas_Estimadas` |
| Función | `Sum` |

### Rollup: Subtotal en USD

| Configuración | Valor |
|---|---|
| Nombre | `Subtotal_USD` |
| Tipo | Rollup |
| Relación | Líneas de Cotización (propiedad `Propuesta`) |
| Campo | `Subtotal_Linea` |
| Función | `Sum` |

### Fórmula: I.V.A. 15 %

```
prop("Subtotal_USD") * 0.15
```

### Fórmula: Total Final USD

```
prop("Subtotal_USD") + prop("IVA_15")
```
